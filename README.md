# 🧠 Психодневник — AI-дневник на OpenGradient

Психологический дневник с верифицированным AI. Записывай мысли — получай анализ настроения,
поддержку и советы. AI помнит твои прошлые записи через MemSync.

## Стек
- **Backend**: Python + Flask + OpenGradient SDK
- **Frontend**: Vanilla JS (открыть index.html в браузере)
- **AI**: OpenGradient TEE LLM (GPT-4.1) — верифицированный инференс
- **Память**: MemSync REST API — долгосрочная память через сессии

---

## Быстрый старт

### 1. Получить ключи

**OpenGradient (уже есть — приватный ключ MetaMask + токены $OPG)**

**MemSync API ключ:**
1. Зайди на https://memsync.ai
2. Зарегистрируйся
3. Найди API ключ в настройках

### 2. Установить зависимости

```bash
cd backend
pip install -r requirements.txt
```

### 3. Задать переменные окружения

**Mac/Linux:**
```bash
export OG_PRIVATE_KEY="0x твой_приватный_ключ_metamask"
export MEMSYNC_API_KEY="твой_memsync_ключ"
```

**Windows (PowerShell):**
```powershell
$env:OG_PRIVATE_KEY="0x твой_приватный_ключ_metamask"
$env:MEMSYNC_API_KEY="твой_memsync_ключ"
```

### 4. Запустить backend

```bash
cd backend
python app.py
```

Сервер запустится на http://localhost:5000

### 5. Открыть frontend

Просто открой файл `frontend/index.html` в браузере.

## Что умеет приложение

| Функция | Описание |
|---|---|
| 📝 Запись мыслей | Поле для ежедневных записей |

| 😊 Анализ настроения | Оценка эмоции + балл 1-10 + теги |
| 📊 График настроения | Визуализация последних 7 дней |

| 📖 История | Все записи сохраняются локально |

## API эндпоинты

```
POST /api/entry     — отправить запись, получить ответ AI
GET  /api/profile   — психологический профиль из MemSync
GET  /api/memories  — поиск воспоминаний (?q=запрос)
GET  /api/health    — статус сервера
```

---

## Без MemSync API ключа

Приложение работает и без MemSync — просто без долгосрочной памяти.
AI будет отвечать на каждую запись без контекста прошлых сессий.
